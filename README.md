## eeprom_flasher

Simple FZ app for reading/writing 24 series eeprom

- SDA - pin 15
- SCL - pin 16

and don`t forget pullup resistors!

Based on [Flipper Zero FAP Boilerplate](https://github.com/leedave/flipper-zero-fap-boilerplate)
